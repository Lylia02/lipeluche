# LiPeluche — Boutique de Peluches

Une boutique e-commerce de peluches développée en **React**.
Projet réel développé pour un client.
Connectée à un backend **Node.js** via une API REST.

## Fonctionnalités

- **Page d'accueil** — Chargement des produits depuis l'API Node.js
- **Filtres** — Filtrage par catégorie (Nounours, Animaux, Pokémon, Jeux vidéo)
- **Page détail** — Informations complètes sur chaque produit
- **Panier** — Ajout, suppression et modification des quantités
- **Récapitulatif** — Total automatique avec livraison gratuite dès 50€
- **Commande** — Envoi au backend via requête POST

## 🛠️ Technologies

- React 18 + Vite
- React Router DOM
- Fetch API

##  Lancer le projet
 Le backend doit tourner en même temps !

```bash
npm install
npm run dev
```

Ouvre `http://localhost:5174`
